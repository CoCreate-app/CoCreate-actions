CoCreate-action

A convenient chain handler allows user to chain multiple CoCreate components together. When one action is complete next one will start. The sequence goes untill all actions completed. Grounded on Vanilla javascript, easily configured using HTML5 data-attributes and/or JavaScript API.

Install

npm install  @cocreate/action

Or you can use cdn link:

https://cdn.cocreate.app/latest/cocreate.min.js
